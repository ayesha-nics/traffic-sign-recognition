# Traffic Sign Recognition using Few-Shot Learning

## 📌 Overview
This project implements a Traffic Sign Recognition system using Transfer Learning (MobileNetV2).
The model is trained on a small dataset to classify traffic signs.

## 🚀 Features
- Image classification using deep learning
- Transfer Learning approach
- Few-shot learning implementation

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## 📂 Classes
- horn
- no_entry
- speed_limit
- stop

## ▶️ How to Run
1. Install dependencies:
   pip install tensorflow matplotlib numpy opencv-python

2. Run:
   python main.py

## 📊 Output
Example:
Predicted Sign: speed_limit

## 🎯 Conclusion
This project demonstrates traffic sign recognition using AI with limited data.
